# Glosář cukrářských termínů (referenční)

> Pro kontrolu, zda garantka používá termíny správně a **jednotně** (vazba na `S-001`, `DOR-S-001`). Ověřeno vícezdrojově (Valrhona, King Arthur, Le Cordon Bleu, CZ weby). [POZOR] = regionálně/úzově sporné — hlídat konzistenci, neoznačovat za chybu.

## Krémy / náplně
| Termín | Definice | Častá záměna |
|---|---|---|
| **Crème anglaise** | mléko+žloutky+cukr, vařeno do 82–85 °C, **bez škrobu** (řídká, lije se) | ≠ pâtissière (ta má škrob); „zhoustnutí" = sražené žloutky (>85 °C) |
| **Crème pâtissière** (pudinkový) | mléko+žloutky+cukr+**škrob**, vařeno do varu | ≠ anglaise |
| **Crème mousseline** | pâtissière + **máslo** (pevná, drží tvar) | ≠ diplomate (ta má šlehačku) |
| **Crème diplomate** | pâtissière + **želatina** + **šlehačka** (lehká) | ≠ mousseline; ≠ bavaroise (jiná báze) |
| **Crème bavaroise** | **anglaise** + želatina + šlehačka | báze anglaise, ne pâtissière |
| **Ganache** | čokoláda + tekutina (emulze tuků) | ≠ crémeux |
| **Ganache montée** | ganache s víc smetanou, odležet → **vyšlehat** | šlehat až vychlazenou, ne za tepla |
| **Crémeux** | (obvykle) **anglaise + čokoláda → žloutky**, hustý, drží tvar | ≠ ganache (crémeux má žloutky, bohatší) |
| **Namelaka** | mléko+čokoláda+**želatina**+smetana, **bez vajec**, hedvábná | ≠ crémeux (namelaka bez žloutků); špatně se mrazí |
| **Pâte à bombe** | **žloutky** + horký sirup (báze pěn; pasterizuje) | ≠ italský sníh (ten z **bílků**) |

## Korpusy / těsta
| Termín | Definice | Záměna |
|---|---|---|
| **Génoise** | piškot ze **šlehaných celých vajec** (teplá metoda) | ≠ biscuit; sušší → punčuje se |
| **Biscuit** (piškotová hmota) | **oddělené** bílky/žloutky, vzdušnější | ≠ génoise; FR *biscuit* zde ≠ sušenka |
| **Dacquoise** | ořechový **sníh**, bez mouky → bezlepkový, křupavý | ≠ joconde (ta má vejce+mouku) |
| **Joconde** | mandle + celá vejce + sníh + mouka, **ohebný** | ≠ dacquoise |
| **Financier** | mandle + bílky (nešlehané) + mouka + **beurre noisette** | hutnější, bez sněhu |
| **Pâte sucrée** | sladké, **crémage** (vejce), pevné → tarty | ≠ sablée |
| **Pâte sablée** | víc másla/žloutků → **nejdrobivější** | ≠ sucrée |
| **Pâte brisée** | bez cukru, studená voda, lámavé → slané/quiche | rozdíl brisée/sablée/sucrée = cukr + metoda |
| **Pâte à choux** | odpalované (vysušit panade, pak vejce), kypří **párou** | vejce do horké panade = sraženina |

## Čokoláda / cukr
| Termín | Definice | Záměna |
|---|---|---|
| **Temperování** | řízená krystalizace kakaového másla (lesk, lupnutí); dark ~45→28–29→31–32 °C | bez něj čokoláda **šedne (bloom)**, matná |
| **Praliné** | pasta z pražených ořechů karamelizovaných s cukrem | **≠ pralinka** (to je bonbon); [POZOR] CZ plete s griliášem |
| **Gianduja** | čokoláda + ~30 % ořechové pasty, vždy **hladká** | ≠ praliné (gianduja má čokoládu) |
| **Croquant / krokant** | karamel + ořechy, lámavé; cukr:ořechy ~1:1 | [POZOR] viz griliáš |
| **Griliáš** (CZ) | karamel + ořechy; [POZOR] CZ **nejednotné** — synonymum krokantu, část zdrojů rozlišuje poměrem | brát jako blízká synonyma |
| **Glaçage miroir** (zrcadlová poleva) | lesklá poleva se želatinou, ~35–37 °C na **zmražený** povrch | ≠ nappage |
| **Nappage** | tenký lesklý gel (meruňka/pektin) na ovoce/tarty, ~50–60 °C | ≠ glaçage (jiný účel) |
| **Karamel suchý vs mokrý** | suchý (cukr sám, rychlý, snáz spálí) vs mokrý (+voda, nemíchat po varu) | mokrý: míchání po varu → zkrystalizuje |

## Techniky
| Termín | Definice |
|---|---|
| **Macaronage** | vmíchání suchých do sněhu u makronek do „lávy"; přemícháno = nelze zachránit |
| **Sablage** | studené máslo vetřít do mouky (drobenka) → křehčí, méně lepku |
| **Crémage** (creaming) | utřít změklé máslo s cukrem → jemnější, kompaktnější |
| **Ruban** (stuha) | vejce/žloutky+cukr vyšlehané tak, že padají v souvislé stuze (dostatek vzduchu) |
| **Foncer / chemiser / masquer** | vyložit formu těstem / vyložit papírem-tukem / potáhnout dort krémem |

## CZ názvosloví — sjednocení (vazba S-001)
- **pusinka / sníh / meringue / bezé** = synonyma pro bílky+cukr; uvnitř materiálu držet **jeden** termín. 3 druhy meringue: francouzská / italská / švýcarská.
- **poleva** = zastřešující; **fondán polevový** (lesklá poleva, věnečky) ≠ **fondán potahovací** (rolovaná hmota, bez mandlí) ≠ **marcipán** (s mandlemi); **glazura** = obvykle royal icing.
- **korpus** = upečené tělo dortu (i nepiškotové) · **piškot** = konkrétní šlehaná hmota. Korpus ⊋ piškot.
- **makronka ≠ kokoska ≠ laskonka** (viz `kategorie/makronky.md`).
